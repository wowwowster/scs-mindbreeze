1 - Restart Connectors
  1. Go to SCS Web Application manager, tab Connectors and click on Monitor for connector instance to be restarted
  2. You will be displayed with connector instance information and buttons which helps you managing it. Click on Stop button, which will stop connector. If stopping process is too long, you can kill it with Kill button.
  3. Restart connector by clicking Start button.
2 - Restart SCS
Sometimes, for updating or maintining SCS, it could be usefull to stop and start whole SCS. This action could only be done with access to the machine on which SCS is intalled.
2.1 - SCS installed on Windows server
  1. Go to SCS installation folder and double click on Setup.exe
  2. Go to Service tab. This menu allows you to Start, Stop and Restart SCS here.
2.2 - SCS installed on Windows server
  1. Go to SCS installation folder, and launch setup.sh
  2. Select Service on setup menu by selecting option 4
  3. You can now Start, Stop or Restart SCS
